10 Web Development Terms

By: Insha Sayani

  1. Node.js

    2. Node.js is an open source environment that minimizes the web server's wait time by continually handling requests without waiting for the server to return content to the client.


  2. React

    3. React is a UI library mostly used for building User Interface components. React is a JavaScript library, and was created by Facebook.


  3. Python

    4. Python is one of the most popular programming languages out there. Since it is a general-purpose language, it is versatile and has a lot of beginner friendliness.


  4. Ruby on Rails

    5. Ruby on Rails or Rails is a MVC framework that is written in the program language Ruby. Rails provides default structures for databases, web services, and web pages.


  5. Sass

    6. Sass is an extension of CSS and stands for Syntactically Awesome Stylesheet. Sass is esentially a pre-processor for CSS and adds in features like variables to make stylesheets easier to maintain.


  6. MySQL

    7. MySQL is a database system that can be used on a web or run on a server. It uses standard SQL and is fast, reliable and easy to use. This makes it ideal for both small and large applications.


  7. Bootstrap

    8. Bootstrap is the most popular CSS framework for developing responsive and mobile-first websites.


  8. AngularJS

    9. AngularJS is a JavaScript framework. It can be added to any web page through a script tag. AngularJS has Directives and Expressions and these is how AngularJS extends HTML attributes to the page.


  9. JSON

    10. JSON stands for JavaScript Object Notation. The JSON syntax is derived from JavaScript syntax, but JSON is text only. Code for reading/generating JSON data can be written in any language.


  10. MongoDB

    11. MongoDB is a document database that stores data in JSON-like documents. This makes the MongoDB database very flexible and scalable.
